# Dictionary
## Usage
```bash
urbandictionary <word>
dictionaryapi <word>
```

## Installation
```bash
go install github.com/KotonBads/dictionary/cmd/urbandictionary
go install github.com/KotonBads/dictionary/cmd/dictionaryapi
```

## Building
```bash
git clone https://github.com/KotonBads/dictionary.git
cd dictionary
./build.sh
```
